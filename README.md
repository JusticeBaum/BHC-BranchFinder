# Installation Instructions
- Navigate to chrome://extensions in a new tab
  - Alternatively, click the Extensions menu puzzle button and select Manage Extensions at the bottom of the menu.
  - Or, click the Chrome menu, hover over More Tools, then select Extensions.
- Enable Developer Mode by clicking the toggle switch next to Developer mode.
- Click 'load unpacked' and select the directory where you are storing the extension

# Usage
- This extension comes preloaded with the keybaord shortcut "ctrl+shift=Z". A user can add their own by editing the manifest.json
- This extension can be pinned to the toolbar by navigating into the 'extensions' menu located in the top right of the Chrome toolbar and selecting the 'pin' option next to the extension in question.
